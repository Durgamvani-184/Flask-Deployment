# 🌸 Iris Flower Prediction Web App (Flask)

A Machine Learning web application built with **Flask** that predicts the species of an Iris flower based on user inputs.

---

## 🚀 Features

* Simple and clean UI
* Real-time predictions
* Pre-trained ML model (Scikit-learn)
* Lightweight Flask backend

---

## 🛠️ Tech Stack

* Python
* Flask
* Scikit-learn
* HTML/CSS

---

## 📂 Project Structure

Flask-Deployment/

│── screenshots/

│── requirements.txt

---

## ▶️ Run Locally

```bash
git clone https://github.com/Durgamvani-184/Flask-Deployment.git
cd Flask-Deployment
pip install -r requirements.txt
python app.py
```

Open browser:
http://127.0.0.1:5000/
