🌸 Iris Flower Prediction Web App (Flask)

A Machine Learning-powered web application built using Flask that predicts the species of an Iris flower based on user inputs. This project demonstrates how to deploy a trained ML model into a real-world web application.

▶️ Run Locally

Step 1: Clone repository
git clone https://github.com/Durgamvani-184/Flask-Deployment.git

Step 2: Move into folder
cd Flask-Deployment

Step 3: Install dependencies
pip install -r requirements.txt

Step 4: Run application
python app.py

Open browser and go to:
http://127.0.0.1:5000/
